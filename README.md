# CodeAlpha Python Projects

This repository contains three fundamental Python projects demonstrating logic, file handling, and user interaction.

## 📂 Projects Included

### 1. 🎮 Hangman Game
A classic word-guessing game.
- **Features:** Random word selection, attempt limiting, input validation.
- **File:** `src/hangman.py`

### 2. 📈 Stock Portfolio Tracker
Calculates total investment based on stock prices and quantity.
- **Features:** Hardcoded stock prices, file handling (saves to `data/portfolio.json`).
- **File:** `src/stock_tracker.py`

### 3. 🤖 Basic Chatbot
A rule-based conversational bot.
- **Features:** Predefined responses, loop control, exit command.
- **File:** `src/chatbot.py`
