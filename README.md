# CodeAlpha Python Internship Projects

Welcome to my Python Projects repository submitted for **CodeAlpha Internship**.  
This repository contains three fundamental Python projects demonstrating logic, file handling, and user interaction.

## 📂 Projects Included

### 1. 🎮 Hangman Game
A classic word-guessing game.
- **Features:** Random word selection, attempt limiting, input validation.
- **File:** `src/hangman.py`

### 2. 📈 Stock Portfolio Tracker
Calculates total investment based on stock prices and quantity.
- **Features:** Hardcoded stock prices, file handling (saves to `data/portfolio.json`).
- **File:** `src/stock_tracker.py`

### 3. 🤖 Basic Chatbot
A rule-based conversational bot.
- **Features:** Predefined responses, loop control, exit command.
- **File:** `src/chatbot.py`

## 🚀 How to Run

1. Clone this repository:
   ```bash
   git clone https://github.com/Asmajavaid1270/CodeAlpha_PythonProjects.git

2. Navigate to the project folder:
   cd CodeAlpha_PythonProjects

3. Run the main file:
python main.py

4. Or run individual scripts:
python src/hangman.py
python src/stock_tracker.py
python src/chatbot.py
